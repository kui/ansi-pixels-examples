Add an example
----------------

1. Create an example on [ANSI Pixels](https://kui.github.io/ansi_pixels/).
2. Add the title and the ANSI Pixels URL hash to `ansi-pixels.tsv`.
3. `npm run build`
4. Do some git operations.
